# 用户使用手册

## 这是什么软件？
这是一个本地声学 OK/NG 检测工具。它不需要联网，不上传数据到云端。检测结果在本机浏览器中显示。

## 怎样启动？
第一次使用先双击 `install_env.bat`。以后每天使用时双击 `run_app.bat`。

## 单个文件怎么测？
打开“首页 / 单文件检测”，上传一个 WAV 文件，点击“开始检测”。

## 一批文件怎么测？
打开“批量检测”，输入文件夹路径，点击运行。可以勾选递归扫描子文件夹。

## 结果怎么看？
- 正常 OK：可以通过。
- 异常 NG：建议拦截。
- 需要复核 REVIEW：主模型认为 OK，但异常监控觉得声音偏离正常产品，建议人工复核。
- 处理失败 FAILED：文件无法读取或格式不支持。

## REVIEW 怎么处理？
REVIEW 样本建议人工听音、复测或加入新批次校准。如果 REVIEW 在某一天突然变多，可能是设备、环境或产品状态发生漂移。

## 新批次校准
如果有带标签的新批次数据，按 `OK/` 和 `NG/` 文件夹放好，在“新批次校准 / 验证”页面运行验证。


## 如果提示 py 不是内部或外部命令

这表示电脑没有安装 Python Launcher，不一定表示没有 Python。新版安装脚本会继续自动查找 `python`、常见安装目录，并在条件允许时尝试用 `winget` 安装 Python。

如果仍然失败，请让电脑管理员安装 Python 3.11，并确认安装时勾选 `Add python.exe to PATH`。安装完成后重新双击 `install_env.bat`。
